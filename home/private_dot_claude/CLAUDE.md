# Tool Preferences

## Always Allowed Tools
- `actionlint` - GitHub Actions linter for validating workflow files
- Always run `actionlint` when working with .github/workflows/ files
- Use `actionlint` to validate GitHub Actions syntax and best practices
- Always use `rg` (ripgrep) for searching text in files instead of `grep`

## Development Tools
- Prefer `actionlint` over manual GitHub Actions validation
- Run `actionlint` automatically after editing workflow files
